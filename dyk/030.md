If you edit headers in vim, set formatlistpat to recognise header lines so auto-formatting behaves. Example setting for common headers:
setlocal formatlistpat=^\\(Cc\\\|To\\\|From\\\|Bcc\\\|Subject\\\|Reply-To\\\):
Also enable 'n' in formatoptions if you want numbered list formatting behaviour.
